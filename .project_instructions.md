# Ad Launch - Project Instructions

## Overview
Freemium web app for automated ad generation. Users enter a website URL, system analyzes the business via Tombstone OS API, generates 3 Facebook ads with watermarks, requires business email registration to download unwatermarked versions.

## Tech Stack
- Next.js 14, React 18, TypeScript
- Prisma ORM with PostgreSQL
- NextAuth.js v4 (credentials provider)
- Tailwind CSS 3, Framer Motion
- Go High Level API (email sending, CRM contacts)
- Tombstone OS API (mission creation, ad generation)

## Color Scheme
- Primary: #2563EB (blue-600)
- Dark: #1D4ED8, Light: #3B82F6
- Accent: #0EA5E9
- Background: #F8FAFC (light), #0F172A (dark sections)

## Key Features
1. Landing page with URL input form
2. Analysis progress tracker (3 steps)
3. Watermarked preview (3 ads + locked SEO/posting plan)
4. Registration modal (business email only, blocks Gmail/Yahoo/etc.)
5. Email confirmation flow via Go High Level
6. Results page (unwatermarked ads, full SEO, posting plan)
7. User dashboard (history, new analysis)
8. Login/signup with business email validation

## API Routes
- POST /api/analyze - Start analysis, create Tombstone OS mission
- GET /api/mission-status - Poll mission status, extract ads
- POST /api/register - Business email registration + GHL contact + confirmation email
- GET /api/confirm-email - Token verification
- GET /api/analysis/[id] - Fetch analysis details
- GET /api/user/analyses - Authenticated user's history
- POST /api/signup - General signup
- POST /api/auth/login - Login
- /api/auth/[...nextauth] - NextAuth handlers

## Database Models
- User: email, password, confirmed, confirmationToken, freeAdsUsed, paidAdsCount
- Analysis: websiteUrl, missionId, status, results (JSONB), seoData, postingPlan
- Ad: imageUrl, watermarkedUrl, caption, headline, watermarked

## Environment Variables
- DATABASE_URL, NEXTAUTH_SECRET
- GHL_API_TOKEN, GHL_LOCATION_ID
- TOMBSTONE_API_URL

## Email Validation
Blocked domains: gmail.com, yahoo.com, rocketmail.com, hotmail.com, outlook.com, aol.com, mail.com, protonmail.com, icloud.com, yandex.com, live.com, msn.com

## Watermark
CSS-based diagonal "Ad Launch" text overlay on preview images, 50% opacity white text with black shadow.
