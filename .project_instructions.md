

## DataForSEO Live Provider (June 2026)
First LIVE Search Intelligence SEO/SERP provider, wired behind the existing provider abstraction.
- **Files:** `lib/dataforseo-provider.ts` (config, Basic-auth header, location mapping, domain helpers, `normalizeSerpResponse`, `DataForSeoProvider` class), `lib/provider-usage.ts` (`logProviderUsage`, `getDataForSeoStatus`). Registered in `lib/search-intelligence-provider.ts` (`'dataforseo'` in `SearchProviderType` union, `REGISTRY`, `resolveProviderType`). Run engine enhanced in `lib/search-intelligence.ts` (`executeSearchIntelligenceRun` now computes self-domain, persists competitors/paid ads/organic history, logs usage, sets isSandbox; new `runSingleTestSearch`).
- **API routes:** `GET .../search-intelligence/provider-health`, `POST .../search-intelligence/test-run`.
- **UI:** SettingsTab (`search-intelligence-manager.tsx`) has provider `<select>` for `defaultProvider`, a DataForSEO status card (Disabled/Missing credentials/Sandbox/Live + last success/error), and a manual test panel (keyword+location, default "transmission flush"/"Houston, TX", language en) rendering organic/paid/other results.
- **Schema additions (pushed, additive):** `ProviderUsageEvent` model; `isSandbox Boolean @default(false)` on SearchIntelligenceRun, SearchVisibilityObservation, PaidAdObservation; `providerUsageEvents` relation on Business.
- **Env:** `DATAFORSEO_ENABLED`, `DATAFORSEO_API_LOGIN/PASSWORD` (Render only — empty locally so it fails gracefully), `DATAFORSEO_BASE_URL`, `DATAFORSEO_SANDBOX_URL`, `DATAFORSEO_USE_SANDBOX` (true→sandbox URL, else live), `DATAFORSEO_DEFAULT_LANGUAGE_CODE=en`, `DATAFORSEO_REQUEST_TIMEOUT_MS`.
- **Endpoints used:** `/v3/serp/google/organic/live/advanced` (SERP), `/v3/keywords_data/google_ads/search_volume/live` (keyword metrics). Official DataForSEO v3 API only — NO Google scraping/browser automation. Credentials never logged/stored (only credentialsRef label).
- **Compliance held:** weekly automation still OFF (settings.enabled defaults false, no cron/daemon); manual run only. Tests: `__tests__/dataforseo-provider.test.ts` (15 passing, covers the 14 required scenarios).
- Note: a pre-existing unrelated test failure exists in `__tests__/competitor-analysis.test.ts` (ftp:// URL assertion) — not caused by this work.
